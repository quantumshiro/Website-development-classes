HTMLのルール

HTML は、Web ブラウザー上で表示される文書を記述するための言語です。 HTML のルールに従って書かれた文書(HTML ファイル)を、Web ブラウザーでは HTML のルールに従って表示します。 Web ブラウザー上で意図したように Web ページを表示するためには、 HTML のルールを守って記述する必要があります。 HTML の基本的なルールを確認しましょう。

要素とタグ

  • 要素は Web ブラウザーへの命令を意味する。
  • html要素は、HTML文書であることを宣言する。
  • 「<」と「>」で要素名を囲んだものをタグと呼ぶ。
  • 要素は、「<」と「>」で囲まれた開始タグ+内容+「<」と「>」で囲まれた終了タグで構成される。
  • 開始タグを記述した場合、終了タグを書かなけらばならない。一部例外を除き、終了タグのない要素はWebブラウザーに正しく認識されない。
  • 要素の中に別の要素を記述したい場合は、入れ子構造で記述する。

改行・スペース

  • 改行は、Webブラウザー上では無効となる。Webブラウザー上で改行を挿入する場合は次のいずれかの方法を用いる。
    1. 改行を表すbr要素を挿入する
    2. 内容を整形済みのテキストとして表示するpre要素を使用する。
    3. ブラックレベル要素で囲むことで、前後に改行が入る。
  • 全角スペース・タブを連続して記述しても、Webブラウザー上では半角スペース1個とみなされる。全角スペースは記述通りに表示される。
  • たとえば、HTML文書内で書かれた次の文書Aは、Webブラウザー上では文書Bのように表示される。
